Detailed Description

Recieves events from Interactor. Implement, and override virtual OnRecieveEvent methods in order to create an Interaction, or subscribe to public events. Provides common functionality useful to all Interactions such as FirstEnter, LastExit events and keeping track of what buttons and interactors are interfacing with it.
